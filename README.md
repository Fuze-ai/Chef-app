

# Chef Menu App

## Overview
A React Native app for chefs to manage their menu items. The app allows adding, viewing, and removing menu items, with features to calculate average prices per course.

## Features
- **Add Menu Items**: Add dishes with name, description, course, and price
- **View Menu**: Display all menu items in a clean card layout
- **Remove Items**: Remove items directly from the list
- **Course Averages**: Automatically calculate average price for each course (Starter, Main, Dessert, Drink)
- **Filter by Course**: Built-in filtering through course selection
